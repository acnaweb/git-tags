# Git Tags

###
[Version Control](https://dzone.com/articles/git-tags-version-control-made-easy)

## Commands

-----
### Version 1.0.0
``` 
git add .
git commit -m "version 1.0.0"
git tag -a v1.0.0 -m "Releasing version v1.0.0" 
```

> Listing tags
``` 
git tag
``` 
> Tag details
``` 
git show v1.0.0
``` 

> Push to remote
``` 
git push origin v1.0.0
```
